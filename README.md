**Freedom Docs**

The text editor for creatives and students. No AI slop. No ads. No rubbish. Freedom Docs.

**Getting Started**

 - Once the .zip file is downloaded extract it to a folder
 - Enter the folder and open the CLICK-ME.html file in your preferred browser

**About the project**

Freedom Docs is all about an open-source. Pulling away from massive corporations who control your creative works.

Freedom Docs is also meant to be accesible and is built with creatives and students as its end users. 

This means that browser-based, performence oriented processes are prefered over others even if they mean less features come to the end user.

Finally Freedom Docs **must** be built for offline use, you should never have to pay an internet bill for something like a text editor.

**Features**

- Edit .md files
- Open images
- File scyncing
- Memory Auto-save
- Voice-Typing
- Zen mode to just focus and write
- Note-taking mode to write text beside your document
- Word Count
- Bold, Italic, Strikethrough, Underline
- Bookmarking documents
- Dark/Light Mode

**Contributing**

As I use this for my own projects, I know this might not be the perfect approach for everyone out there. If you have any ideas, just open an issue and tell me what you think.

If you'd like to contribute, please fork the repository and make changes as you'd like. Pull requests are warmly welcome.

If your vision of a perfect text editor differs greatly from mine, it might be because your projects are vastly different. In this case, you can create a new file Freedom-Docs-yourplatform.html and create the perfect boilerplate for that.
